<h1>Keyboard Shortcuts Reference</h1>
<p>FinalDoc supports keyboard shortcuts throughout the editor and app. Use these to write and navigate faster without reaching for the mouse.</p>

<h2>Editor Text Formatting</h2>
<table>
<thead><tr><th>Action</th><th>Windows/Linux</th><th>Mac</th></tr></thead>
<tbody>
<tr><td>Bold</td><td>Ctrl+B</td><td>Cmd+B</td></tr>
<tr><td>Italic</td><td>Ctrl+I</td><td>Cmd+I</td></tr>
<tr><td>Underline</td><td>Ctrl+U</td><td>Cmd+U</td></tr>
<tr><td>Strikethrough</td><td>Ctrl+Shift+X</td><td>Cmd+Shift+X</td></tr>
<tr><td>Highlight text</td><td>Ctrl+Shift+H</td><td>Cmd+Shift+H</td></tr>
<tr><td>Inline code</td><td>Ctrl+Shift+C</td><td>Cmd+Shift+C</td></tr>
<tr><td>Subscript</td><td>Ctrl+,</td><td>Cmd+,</td></tr>
<tr><td>Superscript</td><td>Ctrl+.</td><td>Cmd+.</td></tr>
<tr><td>Remove formatting</td><td>Ctrl+Shift+M</td><td>Cmd+Shift+M</td></tr>
</tbody>
</table>
