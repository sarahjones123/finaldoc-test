<h1>Settings Overview</h1>
<p>FinalDoc's Settings section is where you configure everything about your project — branding, access control, integrations, billing, and more. This overview explains how Settings is organized so you can find what you need quickly.</p>

<h2>Opening Settings</h2>
<ol>
<li>Click the <strong>Settings (gear icon)</strong> at the bottom of the left sidebar</li>
<li>Or click any <strong>Settings</strong> link throughout the app</li>
</ol>

<h2>Settings Sections</h2>
<table>
<thead><tr><th>Section</th><th>What It Contains</th></tr></thead>
<tbody>
<tr><td><strong>General</strong></td><td>Project name, description, language, timezone, Notification Channels, Knowledge Bases (workspaces)</td></tr>
<tr><td><strong>Appearance</strong></td><td>Portal Builder, Design Studio, Article Display, Smart Bars, Custom CSS &amp; JavaScript, Labels &amp; Text</td></tr>

<h2>AI Agent (OpenClaw) — Separate Section</h2>
<p>The <strong>AI Agent (OpenClaw)</strong> section appears separately in the sidebar (not inside Settings). It's where you provision and manage your AI messaging agent for WhatsApp, Telegram, Slack, Discord, Teams, and WebChat. See the AI Agent (OpenClaw) article for full details.</p>

<h2>Quick Navigation — Settings Search</h2>
<p>At the top of the Settings page, a search bar lets you find any setting by keyword. Type "custom domain" or "SSO" or "notification" and the relevant setting jumps into view.</p>
<p>Keyboard shortcut: Press <strong>Escape</strong> anywhere in the app to activate the settings search bar.</p>

<h2>Settings Are Project-Specific</h2>
<p>All settings in the Settings section apply to the current project only. If you have multiple projects, switch projects using the project switcher at the top of the left sidebar, then access Settings for that project separately.</p>

<h2>Who Can Access Settings</h2>
<ul>
<li><strong>OWNER</strong> — Full settings access including Billing</li>
<li><strong>ADMIN</strong> — Most settings except Billing and account deletion</li>
<li><strong>EDITOR</strong> and below — Cannot access Settings</li>
</ul>

<h2>Project Info</h2>
<p>Your project's basic information — name, description, and KB slug — is in <strong>Settings → General → Project Info</strong>. This is also where you can delete the project (Owners only).</p>
