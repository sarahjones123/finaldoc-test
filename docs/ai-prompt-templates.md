<h1>AI Prompt Templates</h1>
<p>AI Prompt Templates let you save custom instructions that the Writer AI uses when performing actions in the editor. Instead of the default "Improve this writing" prompt, you can create a template that says "Improve this writing using an active voice, max 150 words, suitable for a technical developer audience." Save these once and reuse them across all articles.</p>

<h2>Opening AI Prompt Templates</h2>
<ol>
<li>Go to <strong>Settings → AI (Ved AI) → AI Prompt Templates</strong></li>
</ol>

<h2>Creating a Template</h2>
<ol>
<li>Click <strong>+ New Template</strong></li>
<li>Give the template a <strong>name</strong> (e.g., "Improve for Developers", "Shorten to 100 words")</li>
<li>Choose an <strong>action type</strong> (what Writer AI action this template applies to):
  <ul>
    <li>Improve Writing</li>
    <li>Expand Content</li>
    <li>Summarize</li>
    <li>Custom (free-form prompt)</li>
  </ul>
</li>
<li>Write the <strong>prompt</strong> — the actual instruction sent to the AI. Be specific about tone, length, audience, format</li>
<li>Click <strong>Save Template</strong></li>
</ol>

<h2>Using a Template</h2>
<p>After creating templates, they appear in the Writer AI dropdown in the editor:</p>
<ol>
<li>Select text in the editor (or position cursor in a paragraph)</li>
<li>Click the <strong>AI wand icon</strong> in the toolbar</li>
<li>Choose your custom template from the list</li>
<li>The AI rewrites the text using your custom prompt</li>
</ol>

<h2>Example Templates</h2>
<table>
<thead><tr><th>Template Name</th><th>Prompt</th></tr></thead>
<tbody>
<tr><td>Developer Tone</td><td>Rewrite this for a technical developer audience. Use precise technical language, active voice, and short paragraphs. Avoid marketing fluff.</td></tr>

</tbody>
</table>

<h2>Editing and Deleting Templates</h2>
<p>Click the <strong>pencil icon</strong> next to a template to edit it, or the <strong>trash icon</strong> to delete it. Templates are project-specific — they apply to all team members on this project.</p>
