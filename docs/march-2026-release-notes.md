<h1>March 2026 — Polish &amp; Pricing</h1>
<p>March focused on UI/UX polish, fixing light mode across all pages, reorganizing the sidebar and integrations, and launching the pricing system.</p>
<h2>New Features</h2>
<h3>What's New Panel</h3>
<p>A Gift icon in the top navigation bar shows the latest product updates. Click it to see a timeline of recent changes with tags (New, Improved, Fixed) and links to documentation.</p>
<h3>Task Management Enhancements</h3>
<ul>
<li>Human-readable Task IDs (F-001 for feedback, D-001 for documents, T-001 for translations)</li>
<li>Type filter dropdown to filter tasks by category</li>
<li>Clickable column headers for sorting (Title, Type, Priority, Status, Due)</li>
<li>Scrollable task list and Kanban board for large task counts</li>
</ul>
<h3>Doc Autopilot: Multi-Platform</h3>
<p>Doc Autopilot now supports GitHub, GitLab (including self-hosted), and Bitbucket. Test connection validates all three platforms with descriptive error messages.</p>
<h3>Unified AI Configuration</h3>
<p>Three scattered AI settings pages consolidated into a single AI Configuration page. Supports FinalDoc AI (cloud) or Your Private Cloud (Azure OpenAI, AWS Bedrock, Google Vertex AI, custom endpoint).</p>
<h2>Improvements</h2>
<h3>Sidebar Reorganized</h3>
<p>Clear section groups: Content (Home, Documentation, API Docs, Toolbox, Drive, External Widget), Insights (Analytics, Feedback, Ticket Analysis), AI Studio (Doc Autopilot, Video Studio).</p>
<h3>Integrations: 7 Dedicated Tabs</h3>
<p>Analytics, Chat &amp; Support (with AI Agent), Marketing, Automation (with Webhooks), Code Repositories, Translation, Custom Code.</p>
<h3>Right-Side Slider Panels</h3>
<p>All settings forms now open as right-side slider panels instead of centered popups.</p>
<h3>Feedback System Overhaul</h3>
<ul>
<li>Article Feedback, Ved AI, and Portal Issues tabs</li>
<li>Action buttons: Edit Article, View Article, Create Task, Resolve, Delete</li>
<li>Portal helpful votes now appear in admin Feedback inbox</li>
<li>Task creation from feedback with priority auto-detection</li>
</ul>
<h3>Light Mode Fixes</h3>
<p>Fixed dark-only styles across Settings, Doc Autopilot, Ticket Analysis, Feedback, Analytics, Drive, and Widget Builder.</p>
<h2>Bug Fixes</h2>
<ul>
<li>API Docs: JSON textarea fields now editable (was snapping back on every keystroke)</li>
<li>PDF Export: Images now render correctly (relative URLs rewritten to absolute)</li>
<li>Image alignment standardized across editor, public KB, and PDF</li>
<li>Feedback navigation links fixed (was using broken URL paths)</li>
<li>Video Studio: Create Video button made clickable (was behind background layers)</li>
</ul>
<h2>Documentation</h2>
<ul>
<li>184 published articles across 22 categories on docs.finaldoc.io</li>
<li>19 articles updated for accuracy (renamed features, updated nav paths)</li>
<li>Release Notes category added to public KB</li>
<li>Linking API Docs to Knowledge Base guide published</li>
</ul>
