<h1>AI Advanced Settings &amp; Model Configuration</h1>
<div style="background:#eff6ff;border-left:4px solid #3b82f6;padding:12px 16px;border-radius:4px;margin:16px 0">
  <strong>This feature has been unified.</strong> Model selection and parameters are now part of <a href="/article/ai-configuration"><strong>AI Configuration</strong></a>. Go to <strong>Settings &gt; AI &gt; AI Configuration</strong> to configure your AI provider and model.
</div>
<p>When using <strong>Your Private Cloud</strong>, you can specify the exact model to use (e.g., <code>gpt-4o</code>, <code>gpt-4o-mini</code>, or your custom deployment name). Quality depends on the model you choose — Azure OpenAI with GPT-4o gives identical quality to FinalDoc AI.</p>
