<h2>Editor Toolbar and Formatting Guide</h2>
<p>The FinalDoc editor has a full-featured toolbar at the top of the writing area. Every formatting option has a button in the toolbar, and most also have keyboard shortcuts so you can stay in the flow of writing. This guide covers every tool available.</p>

<h2>Text Formatting</h2>
<table>
<thead><tr><th>Toolbar Button</th><th>Shortcut (Windows)</th><th>Shortcut (Mac)</th><th>What It Does</th></tr></thead>
<tbody>
<tr><td><strong>B</strong> Bold</td><td>Ctrl+B</td><td>Cmd+B</td><td>Makes selected text bold. Use for key terms and important actions</td></tr>
<tr><td><em>I</em> Italic</td><td>Ctrl+I</td><td>Cmd+I</td><td>Italicizes selected text. Use for titles, emphasis, or new terms</td></tr>
<tr><td><u>U</u> Underline</td><td>Ctrl+U</td><td>Cmd+U</td><td>Underlines selected text</td></tr>
<tr><td><s>S</s> Strikethrough</td><td>Ctrl+Shift+X</td><td>Cmd+Shift+X</td><td>Draws a line through text to indicate removal or deprecation</td></tr>
<tr><td>Highlight</td><td>Ctrl+Shift+H</td><td>Cmd+Shift+H</td><td>Highlights text in yellow. Good for drawing attention to warnings</td></tr>
<tr><td>Subscript</td><td>Ctrl+,</td><td>Cmd+,</td><td>Lowers text below the baseline (e.g., H₂O)</td></tr>
<tr><td>Superscript</td><td>Ctrl+.</td><td>Cmd+.</td><td>Raises text above the baseline (e.g., m²)</td></tr>
<tr><td><code>Code</code> Inline</td><td>Ctrl+Shift+C</td><td>Cmd+Shift+C</td><td>Formats a word or phrase as inline code (monospace font)</td></tr>
<tr><td>Clear Formatting</td><td>Ctrl+Shift+M</td><td>Cmd+Shift+M</td><td>Removes all formatting from selected text, returning it to plain paragraph style</td></tr>
</tbody>
</table>

<h2>Headings and Paragraph Styles</h2>
<p>Use the <strong>Paragraph/Heading dropdown</strong> in the toolbar to set the style of the current line:</p>
<table>
<thead><tr><th>Style</th><th>Shortcut</th><th>Use Case</th></tr></thead>
<tbody>
<tr><td>Heading 1 (H1)</td><td>Ctrl+Alt+1</td><td>Page-level titles. Use once per article, at the very top</td></tr>
<tr><td>Heading 2 (H2)</td><td>Ctrl+Alt+2</td><td>Major section headings (e.g., "Installation", "Configuration")</td></tr>
<tr><td>Heading 3 (H3)</td><td>Ctrl+Alt+3</td><td>Sub-sections within an H2 block</td></tr>
<tr><td>Heading 4 (H4)</td><td>Ctrl+Alt+4</td><td>Fine-grained sub-sections</td></tr>
<tr><td>Paragraph</td><td>Ctrl+Alt+0</td><td>Normal body text — the default style</td></tr>
<tr><td>Preformatted</td><td>—</td><td>Monospace block for showing raw text, config files, or terminal output</td></tr>
</tbody>
</table>

<h2>Lists</h2>
<table>
<thead><tr><th>Type</th><th>Shortcut</th><th>When to Use</th></tr></thead>
<tbody>
<tr><td>Bullet List</td><td>Ctrl+Shift+8</td><td>Unordered items where sequence does not matter</td></tr>
<tr><td>Numbered List</td><td>Ctrl+Shift+7</td><td>Sequential steps, procedures, or ranked items</td></tr>
<tr><td>Checklist</td><td>Ctrl+Shift+9</td><td>Interactive to-do style items with checkboxes</td></tr>
</tbody>
</table>
<p><strong>Indenting lists:</strong> Press <strong>Tab</strong> to indent a list item (create a sub-list), and <strong>Shift+Tab</strong> to outdent.</p>

<h2>Links</h2>
<ul>
<li>Select text, then press <strong>Ctrl+K / Cmd+K</strong> (or click the link button) to insert a hyperlink</li>
<li>In the link dialog, paste a URL or start typing an article title to link to another article in your KB</li>
<li>Check the <strong>"Open in new tab"</strong> option for external links</li>
<li>To remove a link, click on linked text and press the <strong>Unlink</strong> button in the link popup</li>
</ul>

<h2>Images and Media</h2>
<ul>
<li><strong>Upload image</strong> — Click the image button in the toolbar, then choose a file from your computer. It uploads to Drive automatically</li>
<li><strong>From Drive</strong> — Click image button → Drive tab → browse and select an already-uploaded image</li>
<li><strong>Paste from clipboard</strong> — Just paste (Ctrl+V) — if you have an image on clipboard (e.g., from a screenshot tool), it inserts directly</li>
<li><strong>Drag and drop</strong> — Drag an image file from your desktop and drop it anywhere in the editor</li>
<li><strong>Image caption</strong> — After inserting, click the image and look for the caption field below it. Captions help with accessibility and SEO</li>
<li><strong>Image size</strong> — Drag the image handles to resize, or click the image and use the size controls in the toolbar</li>
</ul>

<h2>Tables</h2>
<ul>
<li>Click the <strong>Table</strong> button in the toolbar to insert a table. A grid selector lets you choose the number of rows and columns</li>
<li>Click any cell and type. Press <strong>Tab</strong> to move to the next cell</li>
<li>Right-click any cell to access table options: Insert row above/below, Insert column left/right, Delete row, Delete column, Delete table</li>
<li>The first row is automatically formatted as a header row (bold, grey background)</li>
</ul>

<h2>Code Blocks</h2>
<ul>
<li>Click the <strong>Code Block</strong> button (or press <strong>Ctrl+Shift+K</strong>) to insert a syntax-highlighted code block</li>
<li>Select the programming language from the dropdown at the top-right of the block (e.g., JavaScript, Python, Bash, JSON)</li>
<li>Click inside the block and type or paste code</li>
<li>Readers can click the <strong>Copy</strong> button that appears on hover to copy the code to their clipboard</li>
</ul>

<h2>Callout Blocks</h2>
<p>Callout blocks draw attention to important information. There are four types:</p>
<ul>
<li><strong>Info</strong> (blue) — General additional information: "Note: this feature requires the Pro plan"</li>
<li><strong>Tip</strong> (green) — Helpful suggestions and best practices</li>
<li><strong>Warning</strong> (yellow) — Cautions that could lead to unexpected results</li>
<li><strong>Danger</strong> (red) — Critical warnings about data loss or destructive actions</li>
</ul>
<p>Click the <strong>Callout</strong> button in the toolbar, select the type, and write inside the block.</p>

<h2>Diagrams and Math</h2>
<ul>
<li><strong>Mermaid diagrams</strong> — Insert a diagram block and write Mermaid syntax (flowcharts, sequence diagrams, ER diagrams, Gantt charts)</li>
<li><strong>AI diagrams</strong> — Describe what you want in plain English and the AI generates a Mermaid diagram automatically</li>
<li><strong>Math / LaTeX</strong> — Insert a math block and write LaTeX equations (e.g., <code>E = mc^2</code>). Rendered beautifully using KaTeX</li>
</ul>

<h2>Embeds</h2>
<p>You can embed external content directly inside articles:</p>
<ul>
<li><strong>YouTube videos</strong> — Paste a YouTube URL and it embeds as a playable video</li>
<li><strong>CodePen / JSFiddle</strong> — Paste the embed URL and it shows as an interactive code playground</li>
<li><strong>Figma files</strong> — Paste a Figma share link for an interactive design preview</li>
<li><strong>Google Docs / Sheets / Slides</strong> — Paste the published embed URL</li>
<li><strong>Generic iFrame</strong> — Use the iFrame button to embed any URL that allows embedding</li>
</ul>

<h2>Dividers</h2>
<p>Click the <strong>Horizontal Rule</strong> button (or type <code>---</code> on an empty line and press Enter) to insert a visual divider between sections.</p>

<h2>Undo and Redo</h2>
<ul>
<li><strong>Ctrl+Z / Cmd+Z</strong> — Undo last action</li>
<li><strong>Ctrl+Y / Cmd+Shift+Z</strong> — Redo (undo the undo)</li>
</ul>
<p>FinalDoc maintains a full undo history for the duration of your editing session.</p>
