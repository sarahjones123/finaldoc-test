<h1>January 2026 — Core Platform</h1>
<p>The core documentation platform took shape in January, with the fundamental features needed to create, organize, and publish a knowledge base.</p>

<h2>New Features</h2>

<h3>Documentation Editor</h3>
<ul>
<li>Rich text editor with full formatting (headings, lists, tables, code blocks, images)</li>
<li>Category and article tree with drag-and-drop reordering</li>
<li>Article properties inspector (status, description, SEO, tags)</li>
<li>Auto-save with version history</li>
</ul>

<h3>Public Knowledge Base</h3>
<ul>
<li>Public KB portal with FinalDoc subdomain (yourproject.finaldoc.io)</li>
<li>Article reader with table of contents, breadcrumbs, and navigation</li>
<li>Full-text search across all published articles</li>
<li>Mobile-responsive design</li>
</ul>

<h3>API Documentation</h3>
<ul>
<li>OpenAPI/Swagger specification import (YAML and JSON)</li>
<li>Interactive API explorer with request builder</li>
<li>Code examples in 6 languages (cURL, JavaScript, Python, PHP, Ruby, Go)</li>
<li>Multi-version API documentation support</li>
</ul>

<h3>Settings &amp; Configuration</h3>
<ul>
<li>Project settings (name, slug, URL)</li>
<li>Team management with role-based access (Owner, Admin, Editor, Author, Viewer)</li>
<li>Knowledge base versioning</li>
<li>Basic analytics dashboard</li>
</ul>

<h3>AI Writing Assistant (Ved AI)</h3>
<ul>
<li>14+ writing actions: Draft, Improve, Shorten, Lengthen, Simplify, Elaborate</li>
<li>Tone adjustment (professional, casual, technical, friendly)</li>
<li>Auto-generate tags and SEO descriptions</li>
<li>Summarize articles for previews</li>
</ul>
