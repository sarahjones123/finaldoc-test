<h1>Customizing Widget Appearance</h1>
<p>The FinalDoc help widget is fully customizable — you can match it to your product's brand colors, choose button position, set the widget name, customize the icon, and more. All visual settings are in the External Widget's Design tab.</p>

<h2>Opening Widget Design Settings</h2>
<ol>
<li>Click <strong>External Widget</strong> in the left sidebar</li>
<li>Click your widget name</li>
<li>Go to the <strong>Design</strong> tab</li>
</ol>

<h2>Colors</h2>
<ul>
<li><strong>Primary Color</strong> — Used for the floating button background, header bar, and search highlights. Set this to match your brand's primary color</li>
<li><strong>Button text color</strong> — The icon/text color on the floating button (white or dark)</li>
<li><strong>Panel background</strong> — Background of the opened widget panel (white by default; change to match a dark-mode product if needed)</li>
</ul>

<h2>Button Appearance</h2>
<ul>
<li><strong>Button position</strong> — Bottom-right (default), Bottom-left, or Bottom-center of the screen</li>
<li><strong>Button style</strong>:
  <ul>
    <li><strong>Icon only</strong> — A circular button with a help icon (question mark or chat bubble)</li>
    <li><strong>Icon + Label</strong> — Icon with a text label (e.g., "Help") — larger, more noticeable</li>
  </ul>
</li>
<li><strong>Button icon</strong> — Choose from: chat bubble, question mark, document, or upload a custom SVG icon</li>
<li><strong>Button size</strong> — Small, Medium (default), or Large</li>
</ul>

<h2>Widget Panel</h2>
<ul>
<li><strong>Widget name</strong> — Displayed in the panel header (e.g., "Help Center", "Support", "Docs")</li>
<li><strong>Widget subtitle</strong> — A brief tagline shown under the name (e.g., "Search for answers or chat with our AI")</li>
<li><strong>Greeting message</strong> — Message shown before the reader types a search query</li>
<li><strong>Panel width</strong> — Compact (300px), Normal (360px), or Wide (420px)</li>
</ul>

<h2>Dark Mode</h2>
<p>Toggle <strong>Dark Mode</strong> to give the widget a dark background and light text. Ideal for products with a dark-mode first interface. You can also set the widget to <strong>Auto</strong> — it matches the reader's operating system preference (light or dark mode).</p>

<h2>Logo / Branding</h2>
<p>Upload your company logo to display it in the widget header instead of (or alongside) the widget name. Accepted formats: PNG, SVG. Recommended size: 120×40px or similar horizontal logo.</p>

<h2>Preview</h2>
<p>A live preview of the widget appears on the right side of the Design tab. Every change you make is reflected in the preview immediately — no need to save first to see how it looks.</p>

<h2>Saving Changes</h2>
<p>Click <strong>Save Widget</strong> when you're happy with the design. Changes go live within a few minutes on all websites where the widget is installed.</p>
