<h1>File Previews &amp; Management</h1>
<p>Files stored in FinalDoc Drive can be previewed directly in the browser — no need to download them first. This works for images, PDFs, and common document formats. You can also manage file metadata (rename, add descriptions, organize into folders) from the Drive interface.</p>

<h2>Previewing Files</h2>
<p>Click any file in Drive to open the preview panel:</p>
<ul>
<li><strong>Images</strong> (JPG, PNG, GIF, SVG, WebP) — Full-size image preview</li>
<li><strong>PDFs</strong> — Multi-page PDF viewer with zoom controls and page navigation</li>
<li><strong>Videos</strong> (MP4, WebM) — Inline video player</li>
<li><strong>Text files</strong> (TXT, CSV, MD) — Text content preview</li>
<li><strong>Other file types</strong> — Show file info (name, size, upload date) with a download button</li>
</ul>

<h2>File Information Panel</h2>
<p>When a file is selected, the right panel shows:</p>
<ul>
<li><strong>File name</strong> — Click to rename</li>
<li><strong>File type and size</strong></li>
<li><strong>Upload date</strong> and <strong>uploaded by</strong> (team member name)</li>
<li><strong>Public URL</strong> — The CDN link for this file (copy to embed in articles or share externally)</li>
<li><strong>Description</strong> — Optional text description you can add for internal reference</li>
<li><strong>Tags</strong> — Optional tags to help search for this file later</li>
</ul>

<h2>Renaming Files</h2>
<ol>
<li>Click the file in Drive</li>
<li>Click the file name in the info panel (or right-click → Rename)</li>
<li>Type the new name (keep the file extension, e.g., .png, .pdf)</li>
<li>Press Enter or click the checkmark to save</li>
</ol>

<h2>Moving Files to Folders</h2>
<ol>
<li>Right-click the file → <strong>Move to folder</strong></li>
<li>Select the destination folder from the dropdown</li>
<li>Click <strong>Move</strong></li>
</ol>
<p>Or drag and drop the file onto a folder in the left sidebar.</p>

<h2>Deleting Files</h2>
<p>Right-click a file → <strong>Delete</strong>. You'll be warned if the file is embedded in any articles — deleting it will cause broken image placeholders in those articles. Consider replacing the file before deleting.</p>

<h2>Bulk Operations</h2>
<p>Click the checkbox in the top-left of any file card to enter multi-select mode. Select multiple files, then use the bulk action bar to:</p>
<ul>
<li>Move to folder (all selected files move together)</li>
<li>Delete (bulk delete with one confirmation)</li>
<li>Make public / Make private</li>
</ul>
