<h1>Page Templates</h1>
<p>Page Templates let you create article blueprints with pre-filled structure — headings, sections, placeholder text, and formatting guidelines. When a writer starts a new article from a template, they get a consistent starting point instead of a blank page. This saves time and ensures all articles in a category follow the same structure.</p>

<h2>Opening Page Templates</h2>
<ol>
<li>Go to <strong>Toolbox</strong> in the left sidebar</li>
<li>Under <strong>Content Blocks</strong>, click <strong>Page Templates</strong></li>
</ol>

<h2>Creating a Template from Scratch</h2>
<ol>
<li>Click <strong>+ New Template</strong></li>
<li>Give the template a <strong>name</strong> (shown to writers when they pick a template, e.g., "How-to Guide" or "Feature Reference")</li>
<li>Optionally add a <strong>description</strong> explaining when to use this template</li>
<li>Write the template content in the editor. Use placeholder text like "[Describe the feature here]" or "[Add screenshot]" to guide writers</li>
<li>Click <strong>Save Template</strong></li>
</ol>

<h2>Creating a Template from an Existing Article</h2>
<ol>
<li>Open any article that has the structure you want to use as a template</li>
<li>Click the three-dot menu in the editor toolbar</li>
<li>Choose <strong>Save as Template</strong></li>
<li>Give the template a name and save it</li>
</ol>

<h2>Using a Template When Creating an Article</h2>
<ol>
<li>In Documentation view, click <strong>+ New Article</strong></li>
<li>Instead of starting with a blank editor, you will see a prompt asking if you want to use a template</li>
<li>Click <strong>Choose Template</strong></li>
<li>Browse the list of available templates (including FinalDoc's built-in starter templates)</li>
<li>Click a template to preview its structure</li>
<li>Click <strong>Use This Template</strong></li>
<li>The article editor loads with the template content already filled in</li>
</ol>

<h2>Built-In Starter Templates</h2>
<p>FinalDoc includes 12 ready-to-use templates you can import into your project:</p>
<ul>
<li>How-to Guide</li>
<li>Troubleshooting Article</li>
<li>Feature Overview</li>
<li>API Endpoint Reference</li>
<li>Release Notes</li>
<li>Getting Started Guide</li>
<li>FAQ Article</li>
<li>Installation Guide</li>
<li>Concept Explanation</li>
<li>Changelog Entry</li>
<li>Best Practices Guide</li>
<li>Policy Document</li>
</ul>
<p>To import a starter template, click <strong>Import Starter Template</strong> in the Templates view and choose the ones you want.</p>

<h2>Editing and Deleting Templates</h2>
<ul>
<li>Click any template in the list to edit its content or name</li>
<li>Templates are private — only team members with admin access to the Toolbox can create or edit them</li>
<li>Click the trash icon to delete a template (existing articles created from this template are not affected)</li>
</ul>
