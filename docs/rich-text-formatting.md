<h1>Rich Text Formatting</h1>
<p>FinalDoc's article editor is a rich text editor — you write like in a word processor, not in code. This article covers all the text formatting options available and how to apply them.</p>

<h2>Basic Text Formatting</h2>
<table>
<thead><tr><th>Format</th><th>Keyboard Shortcut</th><th>Toolbar Button</th></tr></thead>
<tbody>
<tr><td><strong>Bold</strong></td><td>Ctrl+B / Cmd+B</td><td><strong>B</strong> button</td></tr>
<tr><td><em>Italic</em></td><td>Ctrl+I / Cmd+I</td><td><em>I</em> button</td></tr>
<tr><td><u>Underline</u></td><td>Ctrl+U / Cmd+U</td><td><u>U</u> button</td></tr>
<tr><td><s>Strikethrough</s></td><td>None (use toolbar)</td><td>S button</td></tr>
<tr><td><code>Inline code</code></td><td>Ctrl+backtick / Cmd+backtick</td><td>Code button</td></tr>
</tbody>
</table>

<h2>Headings</h2>
<p>Use headings to structure your article. Heading 1 (H1) is the article title — don't use it in the body. Use H2 for major sections and H3 for sub-sections.</p>
<ul>
<li><strong>Heading 1</strong> — Ctrl+Alt+1 (or type <code>#</code> then Space at the start of a line)</li>
<li><strong>Heading 2</strong> — Ctrl+Alt+2 (or type <code>##</code> then Space)</li>
<li><strong>Heading 3</strong> — Ctrl+Alt+3 (or type <code>###</code> then Space)</li>
<li><strong>Heading 4</strong> — Ctrl+Alt+4</li>
<li><strong>Body text</strong> — Ctrl+Alt+0 (returns to normal paragraph)</li>
</ul>

<h2>Lists</h2>
<ul>
<li><strong>Bullet list</strong> — Click the bullet list icon, or type <code>-</code> then Space at the start of a line</li>
<li><strong>Numbered list</strong> — Click the numbered list icon, or type <code>1.</code> then Space</li>
<li><strong>Checklist</strong> — Click the checklist icon (readers can check/uncheck items on the live article)</li>
</ul>
<p>Press <strong>Tab</strong> inside a list to indent/nest a list item. Press <strong>Shift+Tab</strong> to un-indent.</p>

<h2>Links</h2>
<ol>
<li>Select the text you want to link</li>
<li>Click the <strong>link icon</strong> in the toolbar (or press Ctrl+K / Cmd+K)</li>
<li>Enter the URL</li>
<li>Press Enter or click <strong>Apply</strong></li>
</ol>
<p>To remove a link, click on the linked text and click the <strong>unlink icon</strong> that appears.</p>

<h2>Alignment</h2>
<p>Text and block elements (images, headings) can be aligned:</p>
<ul>
<li>Left align (default) — Ctrl+Shift+L</li>
<li>Center — Ctrl+Shift+E</li>
<li>Right — Ctrl+Shift+R</li>
</ul>

<h2>Text Colors and Highlighting</h2>
<p>Select text, then click the <strong>A (text color)</strong> or <strong>highlight</strong> button in the toolbar to apply a color. Choose from the preset brand-friendly color palette or enter a custom hex code.</p>

<h2>Markdown Shortcuts</h2>
<p>FinalDoc supports Markdown-style shortcuts for fast formatting:</p>
<ul>
<li><strong>**bold**</strong> → bold when you close the second **</li>
<li><code>*italic*</code> → italic</li>
<li><code># Heading</code> → H1 (when typed at start of line)</li>
<li><code>---</code> → horizontal rule (divider)</li>
<li><code>&gt; quote text</code> → blockquote</li>
</ul>
