<h1>Uploading &amp; Organizing Files</h1>
<p>FinalDoc Drive is the media storage library for your knowledge base. All images, PDFs, videos, and other file attachments used in your articles are stored here. You can upload files directly, organize them into folders, and access them from the article editor or for direct sharing.</p>

<h2>Uploading Files</h2>

<h3>Method 1: Drag and Drop</h3>
<ol>
<li>Open <strong>Drive</strong> in the left sidebar</li>
<li>Drag files from your computer directly into the Drive window</li>
<li>Release — files upload immediately and appear in the current folder</li>
</ol>

<h3>Method 2: Upload Button</h3>
<ol>
<li>Open Drive</li>
<li>Click <strong>Upload Files</strong> (or the plus button in the top-right)</li>
<li>Select files from your computer (you can select multiple at once)</li>
<li>Files upload and appear in the current folder</li>
</ol>

<h3>Method 3: Upload While Editing an Article</h3>
<p>When you insert an image in the article editor, you can upload directly from the image insertion dialog without leaving the editor. The uploaded image is saved to Drive automatically.</p>

<h2>File Size and Type Limits</h2>
<ul>
<li><strong>Maximum file size</strong>: 50MB per file</li>
<li><strong>Supported types</strong>: Images (JPG, PNG, GIF, WebP, SVG), PDF, Video (MP4, WebM, MOV), Audio (MP3, WAV), Documents (DOCX, PPTX, XLSX), Text (TXT, CSV, MD), Archives (ZIP)</li>
</ul>

<h2>Creating Folders</h2>
<ol>
<li>In Drive, click <strong>New Folder</strong> (folder icon with plus)</li>
<li>Type the folder name</li>
<li>Press Enter</li>
</ol>
<p>Folders appear in the left panel of Drive. Click a folder to navigate into it. Create sub-folders for deep organization.</p>

<h2>Recommended Folder Structure</h2>
<p>A useful Drive organization for a typical knowledge base:</p>
<ul>
<li><strong>Screenshots</strong> — UI screenshots for articles</li>
<li><strong>Diagrams</strong> — Architecture diagrams and flowcharts</li>
<li><strong>Videos</strong> — Tutorial video clips</li>
<li><strong>Logos &amp; Brand</strong> — Company logos, icons</li>
<li><strong>PDFs</strong> — Downloadable guides and documentation PDFs</li>
<li><strong>Shared Media</strong> — Images used across multiple articles</li>
</ul>

<h2>Searching Files</h2>
<p>Use the <strong>Search</strong> bar in Drive to find files by name. The search works across all folders. You can also filter by file type (Images only, PDFs only, etc.) using the type filter above the file grid.</p>

<h2>Storage Usage</h2>
<p>Your Drive storage usage is shown at the bottom of the Drive sidebar. Your plan includes a storage quota (e.g., 5GB on Starter, 20GB on Business). If you approach the limit, you'll see a warning. Upgrade your plan or delete unused files to free space.</p>
