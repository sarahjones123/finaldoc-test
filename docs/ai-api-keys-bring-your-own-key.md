<h1>AI API Keys (Bring Your Own Key)</h1>
<div style="background:#eff6ff;border-left:4px solid #3b82f6;padding:12px 16px;border-radius:4px;margin:16px 0">
  <strong>This feature has been unified.</strong> API key management is now part of <a href="/article/ai-configuration"><strong>AI Configuration</strong></a>. Go to <strong>Settings &gt; AI &gt; AI Configuration</strong> to set up your provider and API keys.
</div>
<p>The AI Configuration page lets you choose between:</p>
<ul>
<li><strong>FinalDoc AI</strong> — Cloud-hosted, no setup needed, best quality</li>
<li><strong>Your Private Cloud</strong> — Full privacy with Azure OpenAI, AWS Bedrock, Google Vertex AI, or any custom OpenAI-compatible endpoint</li>
</ul>
<p>All AI features (Writer, Chatbot, Translation, Doc Autopilot, Design Assistant) route through your chosen provider.</p>
