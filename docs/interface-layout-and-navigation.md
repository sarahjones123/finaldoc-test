<h2>Top Bar</h2>
<p>The top bar provides quick access to commonly used actions and information. Its content may vary depending on the current screen, but the following elements are always available.</p>
<ul>
<li><strong>Project name and Create button</strong> — Create a new article from any screen.</li>
<li><strong>Search bar (center)</strong> — Search across all articles in the project.</li>
<li><strong>View Site</strong> — Open your public knowledge base in a new tab.</li>
<li><strong>Plan badge</strong> — View your current subscription plan such as Free, Starter, Business, or Enterprise.</li>
<li><strong>Theme toggle</strong> — Switch between light and dark mode.</li>
<li><strong>Notifications bell</strong> — View recent activity, feedback, mentions, and review requests.</li>
<li><strong>Profile avatar</strong> — Access account settings, notification preferences, account switching, and sign out.</li>
</ul>
<hr />
<h2>Documentation Editor</h2>
<p>When you open the Documentation module, the screen is divided into three main sections.</p>
<ul>
<li><strong>Left panel</strong> — Displays the category and article tree. You can drag items to reorder them. Use the <strong>+</strong> icon to add categories or articles. Right-click to access options such as duplicate, move, or delete.</li>
<li><strong>Center editor</strong> — This is the main content area where you write and format articles. You can switch between Rich Text and Markdown modes. Use the toolbar at the top for formatting options.</li>
<li><strong>Right panel (inspector)</strong> — Displays article properties such as publish status, description, featured image, SEO settings, author, tags, and scheduled publish date. You can toggle this panel using the toolbar icon.</li>
</ul>
<hr />
<h2>Settings Sections</h2>
<p>Click the <strong>Settings</strong> icon in the sidebar to configure your project. Settings are grouped into the following sections.</p>
<ul>
<li><strong>General</strong> — Manage project information, knowledge bases (versions), notification channels, and billing.</li>
<li><strong>Appearance</strong> — Configure portal builder, design studio, article display, smart bars, custom CSS and JavaScript, and labels.</li>
<li><strong>Domain &amp; SEO</strong> — Configure custom domain, SEO settings, sitemap, article redirects, and cookie consent.</li>
<li><strong>Integrations</strong> — Organized into seven tabs:
  <ul>
  <li><strong>Analytics</strong> — Google Analytics, Mixpanel, Amplitude, Segment, Hotjar, and more.</li>
  <li><strong>Chat &amp; Support</strong> — AI Agent (OpenClaw), help desks (Freshdesk, Zendesk, Intercom), team collaboration (Slack, Teams).</li>
  <li><strong>Marketing</strong> — Mailchimp, HubSpot, ConvertKit, Freshmarketer, VWO, Google Optimize.</li>
  <li><strong>Automation</strong> — Zapier, Make, Slack notifications, and Webhook Subscriptions.</li>
  <li><strong>Code Repositories</strong> — GitHub, GitLab, Bitbucket sync.</li>
  <li><strong>Translation</strong> — Crowdin, Phrase, Chrome extension.</li>
  <li><strong>Custom Code</strong> — Inject custom HTML into specific zones of your KB layout.</li>
  </ul>
</li>
<li><strong>Security &amp; Access</strong> — Manage site visibility, reader management, security headers, IP restrictions, SSO (SAML/OpenID), and JWT authentication.</li>
<li><strong>Data &amp; Compliance</strong> — Manage backups, audit logs, API tokens, read receipts, and import/export.</li>
<li><strong>AI (Ved AI)</strong> — Configure AI provider, style guide, prompt templates, usage dashboard, AI training, and compliance settings.</li>
</ul>

<h2>Keyboard Shortcuts</h2>
<p>Use keyboard shortcuts to improve productivity while working in the editor.</p>
<p>Press <strong>?</strong> anywhere in the editor to open the keyboard shortcuts reference, or refer to the <strong><a href="/article/keyboard-shortcuts-reference">Keyboard shortcuts reference</a></strong> article for the complete list.</p>
<h3>Quick Shortcuts</h3>
<ul>
<li><strong>Ctrl+S / Cmd+S</strong> — Save the current article.</li>
<li><strong>Ctrl+K / Cmd+K</strong> — Insert or edit a link.</li>
<li><strong>Ctrl+/</strong> — Toggle block type such as heading or list.</li>
<li><strong>Escape</strong> — Open the settings search bar from anywhere.</li>
<li><strong>?</strong> — Open the keyboard shortcuts reference.</li>
</ul>
