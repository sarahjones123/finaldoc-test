<h1>Integrations Overview</h1>
<p>FinalDoc integrates with your existing tools to streamline documentation workflows, track reader behaviour, and automate notifications. Navigate to <strong>Settings → Integrations</strong> to browse and configure available integrations.</p>

<h2>Integration Tabs</h2>
<p>Integrations are organized into seven tabs in the Settings sidebar:</p>

<table>
<thead><tr><th>Tab</th><th>What's Inside</th></tr></thead>
<tbody>
<tr><td><strong>Analytics</strong></td><td>Google Analytics, GTM, Mixpanel, Amplitude, Segment, Heap, Hotjar, FullStory, GoSquared</td></tr>
<tr><td><strong>Chat &amp; Support</strong></td><td>AI Agent (OpenClaw), Freshdesk, Zendesk, Intercom, Crisp, LiveChat, Freshchat, Slack, Microsoft Teams, and more</td></tr>
<tr><td><strong>Marketing</strong></td><td>Freshmarketer, Google Optimize, VWO, Zoho PageSense, Mailchimp, HubSpot, ConvertKit</td></tr>
<tr><td><strong>Automation</strong></td><td>Zapier, Make, Slack notifications, and <strong>Webhook Subscriptions</strong></td></tr>
<tr><td><strong>Code Repositories</strong></td><td>GitHub, GitLab, Bitbucket — sync docs with your code repos</td></tr>
<tr><td><strong>Translation</strong></td><td>Crowdin, Phrase, Chrome browser extension</td></tr>
<tr><td><strong>Custom Code</strong></td><td>Inject custom HTML into specific zones of your KB layout</td></tr>
</tbody>
</table>

<h2>Analytics</h2>
<p>Track reader behaviour and measure documentation effectiveness with external analytics platforms. Connect Google Analytics 4, Mixpanel, Amplitude, Segment, and more.</p>
<p>See <strong>Analytics Integrations</strong> for setup instructions for each platform.</p>

<h2>Chat &amp; Support</h2>
<p>Connect help desks, live chat tools, and AI-powered agents to your knowledge base:</p>
<ul>
<li><strong>AI Agent (OpenClaw)</strong> — Deploy an AI agent that makes your KB accessible from WhatsApp, Telegram, Slack, Discord, and more. See <a href="/article/ai-agent-openclaw">AI Agent (OpenClaw)</a> for setup</li>
<li><strong>Help desks</strong> — Freshdesk, Freshservice, Zendesk, Intercom, Salesforce</li>
<li><strong>Team collaboration</strong> — Slack (<code>/finaldoc</code> commands), Microsoft Teams (<code>@FinalDoc</code> mentions), Drift</li>
</ul>
<p>See <a href="/article/chat-support-integrations">Chat &amp; Support Integrations</a> for details on each platform.</p>

<h2>Marketing</h2>
<p>Connect marketing tools for A/B testing, conversion optimization, and email list building:</p>
<ul>
<li><strong>Freshmarketer</strong> — Marketing and conversion optimization</li>
<li><strong>Google Optimize / VWO</strong> — A/B testing on your knowledge base</li>
<li><strong>Mailchimp / HubSpot / ConvertKit</strong> — Sync KB subscribers with your email lists</li>
<li><strong>Zoho PageSense</strong> — User behaviour tracking and heatmaps</li>
</ul>

<h2>Automation</h2>
<p>Automate workflows between FinalDoc and other services:</p>
<ul>
<li><strong>Zapier</strong> — Connect FinalDoc to 5,000+ apps with automated workflows</li>
<li><strong>Make</strong> — Build powerful multi-step automations</li>
<li><strong>Slack</strong> — Get notifications when articles are published or feedback is received</li>
<li><strong>Webhook Subscriptions</strong> — Configure endpoint URLs to receive HTTP POST notifications when events occur (article published, feedback received, reader registered, etc.). See <a href="/article/webhook-events-subscriptions">Webhook Events &amp; Subscriptions</a> for details</li>
</ul>

<p>Sync your documentation with code repositories for docs-as-code workflows:</p>
<ul>
<li><strong>GitHub</strong> — Two-way sync between FinalDoc articles and a GitHub repo</li>
<li><strong>GitLab</strong> — Sync with GitLab repos (including self-hosted instances)</li>
<li><strong>Bitbucket</strong> — Sync with Bitbucket Cloud repositories</li>
</ul>

<h2>Translation</h2>
<p>Manage multilingual documentation with external translation platforms:</p>
<ul>
<li><strong>Crowdin</strong> — Automate translations and keep content in sync</li>
<li><strong>Phrase</strong> — Professional translation management</li>
<li><strong>Chrome Extension</strong> — Access articles from the Chrome browser toolbar</li>
</ul>

<h2>Custom Code</h2>
<p>Inject custom HTML into specific zones of your KB layout — above/below the header, before/after article content, and in the footer. Use this for announcement banners, disclaimers, tracking pixels, or any custom widget.</p>
<p>See <a href="/article/custom-html-webhooks">Custom HTML</a> for details.</p>
