<h1>Browser &amp; System Requirements</h1>
<p>FinalDoc is a web-based application — it runs in your browser with no software installation required. This article specifies which browsers and operating systems are supported for the best experience.</p>

<h2>Admin Interface (Documentation Editor)</h2>
<p>The FinalDoc admin panel (app.finaldoc.io) is supported on:</p>

<h3>Desktop Browsers</h3>
<table>
<thead><tr><th>Browser</th><th>Minimum Version</th><th>Support Level</th></tr></thead>
<tbody>
<tr><td><strong>Google Chrome</strong></td><td>v110+</td><td>Fully supported (recommended)</td></tr>
<tr><td><strong>Mozilla Firefox</strong></td><td>v115+</td><td>Fully supported</td></tr>
<tr><td><strong>Microsoft Edge</strong></td><td>v110+</td><td>Fully supported</td></tr>
<tr><td><strong>Apple Safari</strong></td><td>v16+</td><td>Fully supported</td></tr>
<tr><td><strong>Opera</strong></td><td>v96+</td><td>Supported (Chromium-based)</td></tr>
<tr><td><strong>Internet Explorer</strong></td><td>Any version</td><td>Not supported</td></tr>
</tbody>
</table>

<h3>Operating Systems</h3>
<ul>
<li><strong>Windows</strong> — Windows 10 or 11 (Edge or Chrome recommended)</li>
<li><strong>macOS</strong> — macOS Ventura (13) or newer recommended; Monterey (12) supported</li>
<li><strong>Linux</strong> — Ubuntu 22.04+ or equivalent, Chrome or Firefox</li>
<li><strong>ChromeOS</strong> — Chromebook with Chrome 110+ (admin access only; no native app)</li>
</ul>

<h2>Mobile — Admin Panel</h2>
<p>The FinalDoc admin panel is designed primarily for desktop use. On mobile, the panel is accessible but some features (especially the documentation editor) are not optimized for touch screens. We recommend using a desktop computer for content creation and editing.</p>

<h2>Public Knowledge Base</h2>
<p>The public knowledge base (reader experience) is fully responsive and mobile-friendly. Supported on:</p>
<ul>
<li>All modern mobile browsers (Chrome for Android, Safari for iOS)</li>
<li>iOS 15+</li>
<li>Android 10+</li>
</ul>
<p>The KB is also a Progressive Web App — readers can install it on their home screen on iOS and Android.</p>

<h2>Internet Connection</h2>
<ul>
<li>A stable broadband connection is recommended for the admin panel</li>
<li>Minimum: 5 Mbps for smooth editing, image uploads, and video generation</li>
<li>Voice mode (OpenAI Realtime) requires 10+ Mbps and a stable connection</li>
<li>The public KB works on slower connections; offline reading is possible for previously visited articles</li>
</ul>

<h2>Screen Resolution</h2>
<p>The admin panel is optimized for 1280×800 and larger. The sidebar may feel cramped on screens smaller than 1200px wide. For smaller laptops, try hiding the browser bookmarks bar and using full-screen mode.</p>
